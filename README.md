## Technologies
- [ ] react
- [ ] redux
- [ ] flowtype
- [ ] jest
- [ ] webpack

## Setup

npm i

Create a .env file in root directory of project
API_CONNECTION=https://menternship.herokuapp.com/ (or if you're running the server locally, point to your server)
PORT=3000

// For development mode
npm run dev

// For production mode 
npm start
